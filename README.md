This repo includes small edits to the package `redcapAPI` for personal use. The official package `redcapAPI` is an R interface to REDCap (http://www.project-redcap.org/), and is an actively developed fork of [redcap](https://github.com/vubiostat/redcap), originally created by [Jeffrey Horner](https://github.com/jeffreyhorner).
